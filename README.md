# FL-KubeOps Operator

Custom Kubernetes operator for federated learning client lifecycle management.
Built with Python and kopf (Kubernetes Operator Pythonic Framework).

## Responsibilities
- FLCluster CRD management
- FL client pod deployment and health monitoring
- Round scheduling, failure recovery, model synchronization

## Tech Stack
Python · kopf · Kubernetes · K3s

Active development — Spring 2026
